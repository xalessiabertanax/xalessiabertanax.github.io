Case Study

Haptic ASL Glove

Learning American Sign Language through muscle memory, haptic feedback, and adaptive music

Role: Hardware & Software Engineer
Year: 2024
Duration: 3 months
ArduinoProcessingPure DataFlex SensorsPhysical Computing

01

The Challenge

Learning sign language traditionally requires visual imitation and repetition. We explored whether haptic feedback and background music tempo could influence the learning curve of the ASL alphabet.

Key Pain Points

1Traditional ASL learning relies solely on visual feedback
2No existing tools combine gesture detection with haptic reinforcement
3Different hand sizes require calibration for accurate detection
4Need to test whether multimodal feedback improves retention

02

The Process

01

Hardware Prototyping

Designed a smart glove with 5 flex sensors and a vibration motor on an Arduino microcontroller, with serial communication to Processing.

02

Calibration System

Built a calibration phase: user holds hand open for 3 seconds, then makes a fist for 3 seconds, setting min/max values per individual.

03

Game Design

Created a 'Memory-Learn' game in Processing with 3 levels. Letters are randomized to ensure true learning, not pattern memorization.

04

Evaluation

Tested short-term retention by asking users to perform gestures without visual aid after completing levels. Measured time per level.

03

The Solution

A complete hardware-software system: a smart glove that detects hand gestures in real-time, paired with a memory game that provides haptic and audio feedback.

✓Real-time gesture detection via 5 flex sensors
✓Haptic vibration feedback on correct gestures
✓Adaptive background music via Pure Data synthesis
✓Per-user calibration for different hand sizes
✓3-level progressive learning game with retention testing

04

The Impact

The prototype demonstrated that multimodal feedback (vibration + music) can enhance gesture learning compared to visual-only approaches.
